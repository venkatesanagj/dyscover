To view a slide show

  1. Place all the images you want to view in a new folder.
  2. On the View menu, click Slide Show.
  3. Select that newly created folder.
  4. An all black screen should appear, click the right button to view the first picture.
  5. To view the next slide, click once with the left mouse button.
  6. To view the previous slide, click once with the right mouse button.
  7. To exit the slide show, double-click the left button.

Note

  • The files in the directory will be shown in alphabetical order.
